# Liner Notes

This folder contains the album liner notes, credits, and thank-yous that will appear in the final release.

## Contents

- `credits.md` - Full credits for all contributors
- `thank-yous.md` - Personal thank-yous and acknowledgments
- `album-story.md` - The story behind the album
- `technical-specs.md` - Recording and production specifications
- `lyrics-booklet.md` - Formatted lyrics for physical release

## Format

Liner notes should be written in Markdown for easy formatting and conversion to other formats (PDF, print, etc.).

## Versions

Keep track of:
- Digital streaming version (minimal)
- Physical release version (full)
- Deluxe edition version (extended)
